## Editor
- control + p 向上移动一行(previous
- control + n 向下移动一行(next
- control + a 行首(ahead
- control + e 行尾(end
- Control-H 删除光标前面的字符
- Control-D 删除光标后面的字符
- control + k 删除到行尾
- <del>command + delete 删除到行首</del>
- Only For VS Code
	* command + shfit + k 删除行
	* command + Enter 向下插入行
	* command + shift + Enter 向上插入行
	* command + shift + \ 跳转到匹配的括号
	* command + Shift + L 选中所有与当前选中内容相同部分
	* control + - 后退
	* control + shift + - 前进
	* option + delete 删除前一个单词
